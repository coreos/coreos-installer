# coreos-installer list-stream

## Description

List available images in a Fedora CoreOS stream

## Usage

**coreos-installer list-stream**

## Options

| **--stream**, **-s** *name* | Fedora CoreOS stream [default: stable] |
| **--stream-base-url** *URL* | Base URL for Fedora CoreOS stream metadata |
